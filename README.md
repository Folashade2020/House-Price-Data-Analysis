# House Price Data Analysis

## Project Overview

This project explores a house price dataset to understand the factors that influence property prices and prepare the data for predictive modeling. The project follows a complete data analysis workflow, from data cleaning and exploratory data analysis to statistical analysis and machine learning.

## Objectives

- Load and inspect the dataset
- Clean and preprocess the data
- Perform Exploratory Data Analysis (EDA)
- Conduct univariate and bivariate analysis
- Explore feature relationships using correlation analysis
- Apply descriptive statistics and hypothesis testing
- Identify key factors influencing house prices
- Build and evaluate machine learning models
